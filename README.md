# Catálogo móvil de Cargueros RISMOTOR 2026

Versión mobile-first para publicar en GitHub Pages, Cloudflare Pages o Netlify.

## Archivos

- `index.html`
- `styles.css`
- `script.js`
- `.nojekyll`
- `assets/cargueros/` con las fichas e imágenes optimizadas en WebP.

## Enfoque de diseño

La interfaz está pensada primero para celular:

- Barra superior compacta.
- Hero vertical con imagen destacada.
- Botones grandes para dedos.
- Filtros horizontales tipo app.
- Navegación inferior fija.
- WhatsApp flotante.
- Fichas técnicas en modal tipo hoja móvil.
- Comparador adaptado a tarjetas en celular.

## Publicación en GitHub Pages

Sube todos los archivos descomprimidos a la raíz del repositorio y activa:

`Settings → Pages → Deploy from a branch → main → /root`
